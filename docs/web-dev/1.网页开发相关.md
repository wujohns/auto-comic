# 网页开发相关
这里主要尝试将 sd 应用到网页开发设计中，加快在网页开发设计部分的效率

## 配色摸奖
模型依赖:  
1. sd 2.1 基础模型: https://huggingface.co/stabilityai/stable-diffusion-2-1/tree/main  
1. WEBUI Helper: https://civitai.com/models/2502/webui-helper?modelVersionId=2786  

备注:  
1. 该模型是目前唯一能找到的关于 webui 设计相关的模型  
1. 已经经过测试该模型与 contorlnet 的配合较差(无法准确表达从而用于网页设计)  
1. 基于以上缘故，这里放弃将其用于准确的设计稿生成，而是用于 webui 的配色摸奖  

## 相关模型

## 23/7/25 测试反馈
经过测试得到的信息如下  
1. 目前较难以借助 sd 获取网页设计上的帮助  
1. 矢量图的制作勉强可以进行，但需要微调的细节较多
