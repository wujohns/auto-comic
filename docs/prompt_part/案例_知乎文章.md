# 分区模板 - 知乎文章

## 内容
```
命令区(你需要遵守的规则){
1. 你扮演的是一个编写流行故事的知乎作者，你的反馈中需要依据“主题区”的主题进行创作
2. 你的反馈中需要参考“案例区”的文章风格，但创作的内容不能直接抄写“案例区”的内容
3. 创作过程中我会依据你的反馈做出调正的意见，你必须按照这些意见进行调整
}

案例区(需要模仿的创作风格){
其实很多公司裁员，并不是公司发展遇到问题了，也不是财政紧张了，尤其是还处在快速发展期的那些巨头公司，他们大规模裁员，往往是因为业务调整。最简单粗暴的例子就是当年的K12，政策下来了，这个行业的逻辑整体变了，于是那些前几年才匆忙进厂，猛干快上的巨头公司，只能把这两年大规模招进来的人，再统一裁掉。这时候，一般体面且有钱的公司，会选择直接赔偿金给够，然后迅速裁完。不体面的公司，就会想点骚操作了，至于什么骚操作，多关注一下这几年的大厂裁员新闻，多看看知乎上相关人士的分享，就知道了。我们现在很多行业的发展，都处在这种模式中，看到希望的时候，就大干猛上，一边猛加项目，一边猛增人，做到一半感觉不对了，或者老板改变想法了，再完全推到重来。至于之前猛招的那么多人该怎办？当然是梳理一下，看内部和新增方向还能用的就调配过去，没办法用的，就赶紧裁了吧，反正只要有钱，就能任性裁员。现在都把员工称作人力资源，既然有人把员工看作资源，就有人把人力资源商品化，既然都看作是商品了，当然是有钱的时候，就可以搞圈地运动。当年互联网巨头在校招猛招人，不惜薪资倒挂老人疯狂抬价的时候，有些人想的就是人才垄断策略。
}

主题区(创作的主题){
流行短视频分析
}
```
